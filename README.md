# cstool

> ⚠️ **学习练手项目**：本项目为作者学习 Rust 语言过程中编写的实践作品，功能尚不完善，**不建议在生产环境中使用**，仅供学习交流参考。

## 📦 项目简介

`cstool` 是一个使用 Rust 编写的简易软件源切换工具，旨在帮助用户快速切换不同地区的软件镜像源，提升包管理器的下载速度。

## 💡 灵感来源

本项目的设计思路参考了优秀的开源项目 [chsrc](https://github.com/RubyMetric/chsrc)，在此特别感谢 @ccmywish 大佬的开源贡献与启发！🙏

## 🤖 AI使用

在注释和测试代码的编写使用了ai辅助，程序的本体逻辑全部是自己编写（~所以写的跟屎一样~）。
